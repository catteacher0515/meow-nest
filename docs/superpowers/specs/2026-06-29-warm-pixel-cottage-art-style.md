# 温暖像素小屋风美术规格

## 目标

本规格用于约束“喵窝”的主美术方向，确保后续生成、筛选、拆分和导入资产时保持统一。

当前主风格确定为：

`温暖像素小屋风`

它不是硬核复古像素，也不是纯 3D 玩具风，而是面向手机竖屏应用的治愈系像素小屋：温暖、明亮、可爱、有轻游戏感，并且适合收集、装扮和截图分享。

## 风格定义

一句话定义：

`一个像素化、温暖、可经营、可装扮的小猫房间。`

关键词：
- 温暖
- 像素
- 小屋
- 治愈
- 收集
- 装扮
- 轻闯关
- 生活感
- 可截图

避免关键词：
- 黑暗
- 赛博
- 硬核复古
- 写实
- 厚重 3D
- 重度手游广告感
- 低清脏像素

## 与竞品气质的关系

可以参考《星露谷物语》式的温暖生活感，但不能直接复刻其角色、建筑、UI 或具体素材。

本项目更偏：
- 手机竖屏
- 小猫陪伴
- 单房间经营
- 轻任务反馈
- 家具装扮收集

而不是：
- 农场模拟
- 大地图探索
- 多 NPC 剧情
- 复杂经营系统

## 视觉原则

### 像素精度

像素风要清晰，但不要过度低分辨率。

推荐方向：
- 角色和家具保留像素边缘
- 主要物体轮廓必须清楚
- 细节可以有像素颗粒，但不能糊
- UI 文本要可读，不追求真实像素字体

不推荐：
- 过度马赛克
- 低清噪点
- 难以辨认的小物件
- 为了复古牺牲可读性

### 色彩

整体色调应偏暖、明亮、治愈。

推荐主色：
- 暖木色
- 奶油黄
- 鼠尾草绿
- 暖橙
- 浅棕
- 柔和天空蓝

辅助色可以用于资源和状态区：
- 爱心红
- 金币黄
- 星星橙
- 鱼干蓝

避免：
- 大面积高饱和霓虹色
- 大面积深色背景
- 冷灰科技感
- 单一棕色导致画面发闷

### 光照

光线应有“下午阳光照进小屋”的感觉。

推荐：
- 温暖侧光
- 台灯局部暖光
- 窗外蓝天和绿色植物
- 柔和阴影

避免：
- 高对比硬阴影
- 夜店灯光
- 恐怖或孤独氛围

## 小猫规范

小猫是产品的情绪核心。

基础比例：
- 大头
- 短身
- 圆眼
- 短腿
- 圆润轮廓
- 坐姿稳定

表情方向：
- 温和
- 开心
- 期待
- 被鼓励
- 陪伴感

基础状态优先级：
1. 默认坐姿
2. 开心
3. 期待
4. 困困
5. 惊喜
6. 被夸后的满足

不做：
- 写实猫
- 攻击性表情
- 过度拟人肢体
- 复杂服装导致主角识别度下降

## 房间规范

房间是长期养成和截图传播的主舞台。

基础构成：
- 地板
- 墙面
- 窗户
- 地毯
- 书桌或小柜
- 台灯
- 书架
- 绿植
- 抱枕
- 墙面挂画

房间需要满足：
- 中央区域能放小猫
- 左右两侧能放家具
- 下方能承载商店或装扮栏
- 背景有生活感但不抢主角
- 新家具加入后能明显改变画面

不做：
- 大地图
- 多房间复杂导航
- 透视过深的 3D 空间
- 细节过密导致小屏幕拥挤

## 家具与装饰规范

家具要同时满足两个场景：
- 能放进房间里
- 能作为商店图标单独展示

第一批家具优先级：
1. 台灯
2. 单人椅
3. 小书柜
4. 抱枕
5. 绿植
6. 地毯
7. 小桌
8. 墙画
9. 窗帘
10. 收纳箱
11. 小床
12. 玩具球

每件家具应具备：
- 清晰剪影
- 单独可识别
- 与主色板协调
- 能在小尺寸下看懂
- 有轻微像素高光和阴影

避免：
- 过细装饰
- 颜色过脏
- 与背景融为一体
- 需要复杂遮挡关系才能成立

## UI 规范

UI 可以有轻游戏感，但不能压过陪伴和治愈感。

保留的结构：
- 顶部资源栏
- 当前任务卡
- 闯关入口按钮
- 家具横向选择栏
- 底部导航

推荐 UI 气质：
- 木质或奶油色面板
- 圆角像素边框
- 小图标资源
- 清晰状态反馈
- 轻微浮起感

文字要求：
- 中文必须清晰可读
- 关键按钮不使用过度像素化字体
- 数字资源栏要一眼看清

避免：
- 重度手游充值 UI
- 过多红点
- 复杂弹窗
- 大面积广告感按钮

## 资产生产规则

每个资产应尽量按可拆分方式生产。

建议资产层级：
- 背景墙面
- 地板
- 固定窗户
- 固定大型家具
- 可替换家具
- 小猫本体
- 小猫装扮
- 前景小物件
- UI 面板
- 图标

优先输出格式：
- 设计预览：PNG

## 验收标准

一张新资产如果满足以下条件，就可以进入候选池：
- 放在手机竖屏画面里一眼能看懂
- 和主参考图气质一致
- 色彩不跳出整体房间
- 小尺寸下仍能识别
- 能单独作为商店图标
- 不会让画面显得拥挤或脏

一张资产应被淘汰，如果：
- 更像 3D 渲染而不是像素小屋
- 更像重度手游广告
- 颜色过暗或过艳
- 轮廓不清
- 与小猫或房间主视觉冲突
- 无法拆分或复用

## 当前结论

“喵窝”的主美术风格确定为：

`温暖像素小屋风`

它兼顾：
- 可爱主角
- 房间装扮
- 轻游戏结构
- 截图传播
- 资产量产

后续所有小猫、家具、房间、UI 和宣传图都应以该风格为默认基准。
